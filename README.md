# ml_rnn

Originial paper: Detecting Dementia through Retrospective Analysis of Routine Blog Posts by Bloggers with Dementia, V. Masrani and G. Murray and T. Field and G. Carenini, ACL 2017 BioNLP Workshop, Vancouver, Canada. In the original paper, authors implemented a SVM classifier for Alzheimer detection, given a set of blog posts written by patients and family members. 

This repo used an rnn many-to-one to classify the same dataset whether a sample belongs to Alzheimer's patients or controls.
